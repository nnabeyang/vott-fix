# VoTT-Fix

VoTT-Fix is a tool to fix [VoTT](https://github.com/microsoft/VoTT) files (especially for local storage project).

# Usage
```sh
./vott-fix -target /path/to/target # target directory contains *.vott and *-asset.json
```
